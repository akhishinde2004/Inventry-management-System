# 📦 Inventory Management System (Java)

A **console-based Inventory Management System** developed using **Java and Object-Oriented Programming (OOP)** concepts.
This application allows users to manage products, update stock, perform sales operations, and view real-time inventory through a menu-driven interface.

---

## 🚀 Features

* ✅ Add new products to inventory
* ✅ Update product stock quantity
* ✅ Sell products with stock validation
* ✅ Prevent duplicate product IDs
* ✅ Display complete inventory records
* ✅ Input validation and error handling
* ✅ Menu-driven CLI interface

---

## 🛠️ Tech Stack

* **Language:** Java
* **Concepts Used:**

  * Object-Oriented Programming (OOP)
  * Classes & Objects
  * Encapsulation
  * ArrayList Collection
  * User Input Handling (Scanner)
  * Control Flow & Validation

## 💡 Functional Overview

### Add Product

Creates a new product with unique ID, name, and quantity.

### Update Stock

Increases available product quantity.

### Sell Product

Reduces stock after validating availability.

### View Inventory

Displays all stored products and quantities.
